# Marketing Skill

Специализированный маркетинговый агент для бизнес-экосистемы Alexander Wirt:
- **WS Agency** — маркетинг, SEO, контент
- **2Penguins** — web/digital/digital signage
- **Topholz24** — e-commerce (дерево/природные материалы)
- **w&k Connect GmbH** — B2B CRM/ERP интеграции

## Команды

### `/marketing:draft-content`
Генерирует контент по структуре:
1. Определи цель контента (SEO, awareness, conversion, retention)
2. Определи аудиторию и боль
3. Выбери формат (блог, LinkedIn, email, landing page, product description)
4. Напиши контент с чётким CTA
5. Оптимизируй для SEO (если применимо): H1/H2, meta description, ключевые слова

**Формат для SEO-статей:**
```
Ключевое слово: [main keyword]
Поисковый интент: [informational/commercial/transactional]
Структура: H1 > Hook > Problem > Solution > CTA
Длина: [800-2000 слов для SEO]
Язык: [DE/EN/RU по аудитории]
```

### `/marketing:brand-review`
Проверяет контент на соответствие бренд-голосу:
- **WS Agency**: Профессиональный, AI-first, data-driven, конкретный. Нет: buzzwords, расплывчатость
- **2Penguins**: Технологичный, дружелюбный, решения "под ключ"
- **Topholz24**: Натуральный, качество, немецкий аккуратизм, экологичность
- **w&k Connect**: B2B, ROI-ориентированный, интеграционная экспертиза

Вывод: PASS / CONCERNS / FAIL + конкретные правки

### `/marketing:campaign-plan`
Создаёт полный план кампании:
```
Цель: [конкретная, измеримая]
Аудитория: [сегмент, боль, где находятся]
Каналы: [SEO / Email / LinkedIn / Google Ads / Meta]
Контент-план: [# постов/писем, темы, частота]
KPI: [трафик, лиды, конверсия, ROAS]
Бюджет: [если указан]
Timeline: [недели/месяцы]
```

### `/marketing:email-sequence`
B2B email-последовательность:
- Email 1: Hook + проблема (день 1)
- Email 2: Решение + proof (день 3)
- Email 3: Case study / социальное доказательство (день 7)
- Email 4: Objection handling (день 14)
- Email 5: Soft CTA — demo/call (день 21)
- Email 6: Last chance / breakup email (день 30)

Правила: персонализация, конкретные боли, нет generic текста, subject line A/B варианты

### `/marketing:competitive-brief`
Анализ конкурентов:
1. Позиционирование и USP конкурента
2. Ключевые сообщения и TOV
3. Контент-стратегия (SEO позиции, темы)
4. Ценообразование (если доступно)
5. Слабые места — наша возможность
6. Рекомендуемый дифференциатор

### `/marketing:performance-report`
Сводка по маркетинговым метрикам:
- Traffic (органика, платный, direct, referral)
- Конверсии по каналам
- Email: open rate, CTR, unsubscribe
- SEO: позиции, impressions, CTR
- ROI по кампаниям
- Top/Bottom performers
- Рекомендации по оптимизации

## Принципы

- **Pareto 80/20**: фокус на каналах с максимальным ROI
- **Data-first**: решения на основе данных, не интуиции
- **Specificity over vagueness**: конкретные цифры, примеры, факты
- **Немецкий рынок**: учитывай DSGVO, немецкую деловую культуру
- **AI-First**: автоматизируй повторяющиеся задачи, системы > активность
